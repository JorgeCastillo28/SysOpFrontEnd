# prueba-sysop-front-end

## Project setup
```
-- Añadir archivo .env en la raíz del proyecto con las siguientes variables de entorno --

  VUE_APP_ENV=local
  VUE_APP_API_URL=

npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
